# Site d'e-commerce :

## Présentation du projet :

Développement d'un site d'e-commerce à l'issue d'un projet commencé au sein de l'IUT de Belfort. 
Nous devons, par groupe de 3-4, réalisé à partir d'une maquette donnée, un site d'e-commerce avec pour commencer une gestion d'un formulaire de connexion avec un compte administrateur, un compte client et un deuxième compte client. Les 2 comptes clients servent de tests pour vérifier que tout fonctionne correctement (toutes les fonctionnalités seront présentées plus tard). ##Suite
